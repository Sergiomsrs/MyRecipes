# Documentación

Esta carpeta contiene la documentación técnica y funcional del proyecto **MyRecipes**.

El objetivo de esta documentación es describir el diseño de la aplicación antes de comenzar su desarrollo, facilitando la comprensión de las decisiones tomadas y sirviendo como referencia durante la implementación.

## Estado

La documentación se encuentra en constante evolución y podrá actualizarse conforme avance el desarrollo del proyecto.

Una vez implementadas las distintas funcionalidades, cada documento reflejará tanto el diseño inicial como las decisiones adoptadas durante la implementación.
